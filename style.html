STYLE INFORMATION

style sheets are powerful mechanism for adding styles to web documents.HTML elements on a web page can then be bound to the style sheet.The advantages of a style sheet includes the ability to make global changes to all documents from a single location.


style attributes are....
  • font attributes
  • text attributes
  • color and background attributes
  • margin attriutes
